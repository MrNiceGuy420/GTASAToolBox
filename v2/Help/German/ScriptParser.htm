ScriptParser

Der ScriptParser der GTA San Andreas ToolBox kann selbst erstellte Scripts ausführen. Die Syntax der Programmiersprache ist wie die SCM-Syntax von GTA San Andreas. Sie funktioniert nicht wie eine normale Programmiersprache mit bestimmten Befehlen wie zum Beispiel MessageBox() sondern mit Opcodes. Bei Opcodes wird an jeden Anfang der Zeile ein Code geschrieben. Dieser besteht aus 4 Zeichen, die in Hexadezimal sind. Nach den 4 Zeichen kommt ein Doppelpunkt und dann der erste Parameter der Funktion. Dieser ist oft der Rückgabewert der Funktion. Nach dem Rückgabewert kommt der nächste Parameter. Es können für jede Funktion maximal 100 Parameter + Rückgabewert verwendet werden. Also sind es 101 Parameter.

 

Befehle

Nachrichtenfenster

Opcode: 0001
Parameter: Rückgabewert, Titel, Nachricht, Flags

 

Eingabefenster

Opcode: 0002
Parameter: Rückgabewert, Titel, Nachricht, Standardtext

 

Variablenüberprüfung

Opcode: 0003
Parameter: Variable1, Typ, Variable2

 

Label

Opcode: 0004
Parameter: Name

 

Teleporter

Opcode: 0005
Parameter: x, y, z, Blickwinkel, Interior

 

Wert speichern

Opcode: 0006
Parameter: Name, Wert

 

Fahrzeug spawnen

Opcode: 0007
Parameter: VehicleID

 

Speicheradresse ändern

Opcode: 0008
Parameter: Adresse, Datentyp, Wert, Länge*, Leerzeichen*
*Nur bei Datentyp "STRING"

 

Goto

Opcode: 0009
Parameter: Name

 

Programm ausführen

Opcode: 000A
Parameter: Dateiname, Programmparameter

 

Speicheradresse auslesen

Opcode: 000B
Parameter: Rückgabewert, Adresse, Datentyp, Länge*

*Nur bei Datentyp "STRING"

 

Variable erstellen (ohne Zuweisen eines Werts)

Opcode: 000C
Parameter: Variablenname

 

Text ersetzen

Opcode: 000D
Parameter: Rückgabewert, Text, Suchtext, Ersetztext

 

Funktionsspeicheradresse aufrufen

Opcode: 000E
Parameter: Rückgabewert, Adresse, Funktionsparameter

 

Inhalt einer Datei in Variable speichern (String-Modus)

Opcode: 000F
Parameter: Rückgabewert, Dateiname

 

Inhalt einer Datei in Variable speichern (Memory-Modus)

Opcode: 0010
Parameter: Rückgabewert, Dateiname

 

Speicherbereich freigeben

Opcode: 0011
Parameter: Speicheradresse

 

Dateiauswahldialog anzeigen

Opcode: 0012
Parameter: Rückgabewert, Typ, Titel, Standarddatei, Pattern, Standardpattern

 

In Datei schreiben

Opcode: 0013
Parameter: Rückgabewert, Dateiname, Typ, Daten, Länge*

*Nur bei Typ "DATA"

 

Überprüfen, ob eine Datei vorhanden ist

Opcode: 0014
Parameter: Rückgabewert, Dateiname

 

Taste abfragen

Opcode: 0015
Parameter: Rückgabewert, VK-Taste, Goto-Label*

*Wenn angegeben, wird zu dieser Position gesprungen

 

Warten (In Millisekunden)

Opcode: 0016
Parameter: Zeit

 

Zu Zeile springen

Opcode: 0017
Parameter: Zeilennummer
