Einstellungen

In den Einstellungen der GTA San Andreas ToolBox kannst du Optionen wie z.B. der Dateiname und -Pfad der GTA San Andreas-exe (gta_sa.exe) festlegen, Das Hintergrundbild der Modulfenster ändern oder die Sprache ändern.

Allgemein

Nur eine Instanz starten

Jedes Modul (GUI, CheatServer, WebServer, ...) der GTA San Andreas ToolBox kann nur einmal gestartet werden.

Fenstereinstellungen merken

Die Position und Größe der Modulfenster sowie des Hauptfensters werden gespeichert und beim nächsten Start wiederhergestellt.

In den Systray minimieren

Die GUI wird statt in die Taskleiste in den Systray minimiert.

Minimiert starten

Die GUI wird bei Start minimiert gestartet. Ist die Option "In den Systray minimieren" aktiviert, wird beim Start nur das Trayicon angezeigt.

Nachrichten anzeigen

Nachrichten wie z.B. "Interior 0 geladen" oder "Cheat aktiviert (RIPAZHA)" in GTA San Andreas anzeigen.

Beim Beenden nachfragen

Beim Beenden der GUI wird nachgefragt, ob du sie wirklich beenden möchtest.

Tabs verwenden

Die Fenster der einzelnen Module können zusätzlich zum Menü über die Tabs ausgewählt werden.

Splashscreens deaktivieren

Beim Start von GTA San Andreas werden keine Splashscreens mehr angezeigt (Nur wenn GTA San Andreas über die ToolBox gestartet wird).

Alles beim Beenden schließen

Wird diese Option aktiviert, wird beim Beenden der GTA San Andreas ToolBox GUI auch der CheatServer und der WebServer beendet (Wenn diese gestartet sind).

Cheatsound

Wenn du einen Sound hören möchtest, wenn eine Tastenkombination aus der Cheatliste gedrückt wurde, kannst du den Standardsound hier festlegen.

Wähle die Option "Verwenden" und wähle die Datei aus. Es werden wav und bmf-Dateien unterstützt.

Hintergrundbild

Wenn du ein Hintergrund für den Bereich der Modulfenster haben möchtest, kannst du hier ein Bild auswählen, welches dann als Hintergrund verwendet wird.

Das Bild wird automatisch an die Größe des Fensters angepasst.

Zum Verwenden muss die Option "Verwenden" aktiviert sein.

GTA San Andreas

In diesem Feld kannst du die GTA San Andreas-exe (gta_sa.exe) auswählen. Diese Datei wird ausgeführt, wenn du auf "GTA San Andreas starten" klickst.

CheatServer - Cheats beim Start aktivieren

Hier kannst du die Cheats einfügen, die automatisch beim Start des CheatServers aktiviert werden sollen.

Wähle "Hinzufügen", um einen Cheat hinzuzufügen. Gebe dann den Cheat ein und klicke auf OK. Wird das Feld frei gelassen, wird die Eingabe abgebrochen.

Wähle "Entfernen", um einen Eintrag aus der Liste zu löschen.

Um alle Einträge aus der Liste zu löschen, klicke auf den Button "Liste löschen".

Bildformat

In diesem Bereich kannst du das Bildformat für die Screenshots auswählen. Es stehen 3 Formate zur Verfügung (BMP, JPG und PNG).

User Track Player

Hier kannst du Optionen für den User Track Player der ToolBox GUI festlegen.

Es stehen dir 2 Optionen zur Auswahl:

ID3-Tags beim Start laden

Die ID3-Tags der Trackliste werden beim Start ausgelesen.

ID3-Tags beim Aktualisieren laden

Die ID3-Tags der Trackliste werden beim Aktualisieren der User Track-Liste (Über den "Aktualisieren"-Button) ausgelesen.

Start

In diesem Feld kannst du auswählen, was beim Start der ToolBox GUI gestartet werden soll.

Es stehen dir die 2 Optionen "CheatServer" und "WebServer" zur Verfügung.

Dateitypen

Im Bereich "Dateitypen" kannst du die Beschreibungen der Dateitypen ändern. Diese werden z.B. im IMG Editor angezeigt.

Hinweis: Momentan kann diese Liste noch nicht geändert werden!

WebServer

In diesem Feld kannst du die Optionen des WebServers anpassen.

Port

Der Port des Servers. Dieser Wert muss eine Nummer von 1 bis 65000 sein.

Eine Liste der Standardports findest du hier: Portliste

Der WebServer ist ein HTTP-Server und hat somit den Port 80 als Standard.

Wenn ein anderes Programm auf diesem Port bereits läuft, kannst du auch einen anderen Port verwenden.

Sprache

Hier kannst du die Sprache der GTA San Andreas ToolBox und dieser Hilfe ändern.

Zur Auswahl stehen dir die Sprachen Deutsch und Englisch.
