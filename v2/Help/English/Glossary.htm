Glossar

Auf dieser Seite werden dir wichtige Begriffe erklärt, die möglicherweise unverständlich sind.

 Begriff

 Beschreibung

 Backup  Sicherungskopie von Daten.
 Button  Eine Schaltfläche bzw. ein Knopf in einem Programm.
 CheatServer  Teil der GTA San Andreas ToolBox. Er wird verwendet, um die Tastenkombinationen zu empfangen und die Cheats zu senden.
 Dialog  Ein Fenster mit Buttons zur Auswahl von Dateien, hinzufügen von Einträgen, usw.
 GUI  Grafische Benutzeroberfläche (GUI = Graphical User Interface).
 HTTP  Das Übermittlungsprotokoll für Internetseiten (HTTP = Hyper Text Tranfer Protocol).
 ID3-Tag  Titelinformationen bei Musik (Z.B. Titelname, Interpret oder Album).
 Interior  Ein Bereich (Z.B. ein Haus) in GTA San Andreas.
 Modul  Teil eines Programmes.
 Port  Ein Anschluss an einem Server (Port = Anschluss).
 Screenshot  Foto des Bildschirms.
 Scrollleiste  Eine Leiste, mit dem du durch Dinge durchblättern kannst.
 Separator  Abstandshalter
 Splashscreen  Ein Bildschirm oder ein Fenster, auf dem ein oder mehrere Logos angezeigt werden.
 Systray bzw. Tray  Infobereich in der Taskleiste neben der Uhr.
 Tab  Registerkarte

 Toolbarsymbol

 Ein Symbol in der Toolbar.

 Toolbar

 Leiste mit verschiedenen Symbolen (Wie ein Menü).

 Trayicon

 Symbol im Systray.

 WebServer

 Teil der GTA San Andreas ToolBox. Er wird verwendet, um die Weboberfläche zur Verfügung zu stellen.

 

Hinweis: Die Liste ist möglichweise unvollständig!
