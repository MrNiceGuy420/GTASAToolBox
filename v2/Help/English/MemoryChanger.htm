MemoryChanger

The MemoryChanger is an easy tool of the GTA San Andreas ToolBox to change any memoryaddress of GTA San Andreas.

You can read and write the following datatypes:

 Name

 Size

  Range
 Byte  1 Byte  -128 to +127
 Word  2 Bytes  -32768 to +32767
 Character  2 Bytes  0 to +255
 Long  4 Bytes  -2147483648 to +2147483647
 Float  4 Bytes  Unlimited
 Double  8 Bytes  Unlimited
 Quad  8 Bytes  -9223372036854775808 to +9223372036854775807
 String*  Length of the string + 1 Byte  Unlimited

*Not available yet!

 

Hinzufügen von Speicheradressen

Damit du dir Speicheradressen nicht merken oder irgendwo aufschreiben musst, kannst du die Speicheradressen in eine Liste aufnehmen. In dieser Liste wird der Name der Adresse, die Adresse in Hexadezimal, der Datentyp sowie der aktuelle Wert angezeigt.

Um eine Adresse hinzuzufügen, musst du nur einen Name für die Adresse und die Adresse in Hexadezimal eingeben. Dann noch den Datentyp wählen und auf den Button "Hinzufügen" klicken.

Um etwas an der Speicheradresse zu ändern, musst du die Speicheradresse wählen und die Werte wie beim Hinzufügen ändern. Dann klickst du auf den Button "Übernehmen".

 

Aktualisieren der Werte

Da sich die Werte der Speicheradressen nicht von selbst aktualisieren, musst du auf den Button "Lesen" klicken. Alle Adressen werden dann erneut aus GTA San Andreas ausgelesen und die Werte in der Liste werden aktualisiert.

 

Ändern von einem Wert

Um einen Wert einer Speicheradresse in GTA San Andreas zu ändern, musst du nur die Speicheradresse auswählen (Doppelklick auf einen Eintrag aus der Speicheradressenliste) und auf den Button "Schreiben" klicken.
