ScriptParser Beispiele

Auf dieser Seite findest du Beispielcodes zu den Befehlen des GTA San Andreas ToolBox ScriptParsers. Da in Html allerdings das Tab-Zeichen oft nicht richtig angezeigt wird, verwende ich in dieser Hilfe ein anderes Zeichen. Du musst dieses Zeichen aber beim einfügen in einen Code wieder durch den Tab ersetzen. Am einfachsten geht dies mit der Ersetzfunktion eines Texteditors wie zum Beispiel der Windows Texteditor "Notepad".

Hinweis: Ich verwende in dieser Hilfe [TAB] als Tab-Zeichen.

 

Nachrichtenfenster mit Benutzerdefiniertem Text anzeigen

0002: $Name$ [TAB] Willkommen [TAB] Hallo, bitte gebe deinen Name ein. [TAB] Dein Name
0003: $Name$ [TAB] EQUAL [TAB] [TAB] Error
0001: [TAB] Willkommen [TAB] Hallo $Name$!\n\nSchön, dass du hier bist. [TAB] MB_ICONINFORMATION
0009: Quit
0004: Error
0001: [TAB] Fehler [TAB] Du hast keinen name angegeben! [TAB] MB_ICONERROR
0004: Quit
